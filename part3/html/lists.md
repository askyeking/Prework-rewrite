## HTML Lists

There are two types of lists, *Unordered Lists* (`<ul>`), which use bullet points to the list of defined *List Items* (`<li>`), and *Ordered Lists* (`<ol>`), which use numbers.

### Unordered Lists (`<ul>`)

```html
<h4>Shopping List:</h4>

<ul>
  <li>Milk</li>
  <li>Garlic</li>
  <li>Broccoli</li>
  <li>Carrots</li>
</ul>
```

![Unordered Lists](images/htmlULEx.png)

<hr>

### Ordered Lists (`<ol>`)
```html
<h4>All Time Top Five Favorite Movies:</h4>

<ol>
  <li>The Fast and the Furious</li>
  <li>2 Fast 2 Furious</li>
  <li>The Fast and the Furious: Tokyo Drift</li>
  <li>Fast & Furious</li>
  <li>Fast Five</li>
</ol>
```

![Unordered Lists](images/htmlOLEx.png)

<hr>

### Nested Lists
```html
<h4>Todo List:</h4>

<ol>
  <li>Make the bed.</li>
  <li>Have breakfast.</li>
  <ul>
    <li>Scambled Eggs</li>
    <li>Pancakes</li>
    <li>Coffee</li>
  </ul>
  <li>Contemplate the meaning of life.</li>
</ol>
```
