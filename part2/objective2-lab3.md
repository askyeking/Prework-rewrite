## Lab 3 - Creating and Using Variables

### Overview
* Create a file called _Objective2Lab3.java_ in the _SDPre/Labs_ directory.
* Using the _Starter Code_ below add code to the commented area that would:
  * Assign values of your 3 favorite foods to the provided variables.
  * Print the String 'My Favorite Foods Are:'
  * Once the values are assigned print them out on there own lines.

### Starter Code
```java
public class Objective2Lab3 {
  public static void main(String[] args) {
    String food1, food2, food3;

    /*
      Your solution goes here
    */
  }
}
```
### Expected Input
```bash
>> javac Objective2Lab3.java
>> java Objective2Lab3
```
### Expected Output
```bash
My Favorite Foods Are:
1. Apple
2. Banana
3. Coconut
```
