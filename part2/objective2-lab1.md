## Lab 1 - Printing a Variable

### Overview
* Create a file called _Objective2Lab1.java_ in the _SDPre/Labs_ directory.
* Using the _Starter Code_ below add code to the commented area that would print out the value `"Hello World"` stored in the hello variable.

### Starter Code
```java
public class Objective2Lab1 {
  public static void main(String[] args) {
    String hello = "Hello World";

    /*
      Your solution goes here
    */
  }
}
```
### Expected Input
```bash
>> javac Objective2Lab1.java
>> java Objective2Lab1
```
### Expected Output
```bash
Hello World
```
