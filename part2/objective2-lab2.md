## Lab 2 - Printing Multiple Variables

### Overview
* Create a file called _Objective2Lab2.java_ in the _SDPre/Labs_ directory.
* Using the _Starter Code_ below add code to the commented area that would:   
  * print out your name, age and the state where you were born.
  * Change the values to reflect your own personal information.

### Starter Code
```java
public class Objective2Lab2 {
  public static void main(String[] args) {
    String name = "Charles Peterson";
    int age = 47;
    String state = "Arkansas";

    /*
      Your solution goes here
    */
  }
}
```
### Expected Input
```bash
>> javac Objective2Lab2.java
>> java Objective2Lab2
```
### Expected Output
```bash
Name: Charles Peterson Age: 47 State: Arkansas
```
